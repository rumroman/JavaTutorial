
                    Класс Pattern.
    В классе не определяется доступных конструкторов. Для составления шаблона вызывается
фабричный метод compile().

static Pattern compile(String шаблон)

"шаблон" обозначает регулярное выражение , которое требуется использовать.Метод compile()
преобразует в шаблон символную строку, определяемую параметром "шаблон", для сопоставления
средствами Matcher. Этот метод возвращает обхект типа Pattern, содержащий шаблон.

Как только объект типа Pattern будет получен, его можно использовать для создания объекта
типа Matcher. Для этого вызывется фабричный метод matcher(), определяемый в классе Pattern.

Matcher matcher(CharSequence строка)

"строка" обозначает последовательность символов, сопоставляемую с шаблоном и называемую входной
последовательностью.

                    Класс Matcher.
   Отсутствуют конструкторы. Для создания обхекта класса Matcher вызывается фабричный метод matcher()
определяемый в классе Pattern. Самым простым для сопоставления с шаблоном является метод matches(),
который просто определяет, совпадает ли последовательность символов с шаблоном.

  boolean matches()

С шаблоном должна совпадать вся последоватльность символов.

   Чтобы определить , совпадает ли с шаблоном подпоследовательность из входной полседовательности символов,
следует вызвать метод find()    .

   boolean find()

   Метод find() можно вызывать неоднократно , чтобы находить все совпадающие подпоследовательности. При каждом
вызове метода find() сравнение начинается с того места, где было завершено предыдущее сравнение .

   Символьную строку , содержащую последнюю совпавшую последовательность , можнопо получить ,
вызвав метод group().

   String group()

Этот метод возвращает совпавшую символьную строку.Если ни одного совпадаения не обнаружено, генерируется
исключение типа IllegalStateException.

   Вызвав метод start() , можно получить индекс текущего совпадаения во входной последовательности  , а
индекс , следующий после текущего совпадения - метод end().

  int start()
  int end()

  Каждую совпавшую последовательность символов можно заменить другой последовательностью, вызвав метод
 replaceAll()

   String replaceAll(String новая_строка)
